# Thomas Rees's Electronic Laboratory Notebook
Electronic laboratory notebook for Thomas Rees, a postgraduate taught Master's student at University College London within [The Todd Group](https://todd-lers.github.io/about/), led by [Professor Mathew Todd](https://todd-lers.github.io/about/current.html).

This ELN contains experiments conducted for the [SGC Open Chemistry Network](https://www.thesgc.org/sgc-open-chemistry-networks) A global private public partnership focusing on understanding the functions of all human proteins with an emphasis on open source research to create novel bioactive small molecules and use them to accelarate the search for new medicines. Specifically, these experiments are contributing towards SGC Open Chemistry Network Project number 45 ([CNP45-ABHD14B](https://github.com/StructuralGenomicsConsortium/CNP45-ABHD14B)) which is focussed on finding a probe for Alpha/Beta Hydrolase Domain-containing Protein 14B (ABHD14B). More information on the projects and the contributions store in this ELN can be found in the [Wiki](https://github.com/Tom-QR/ELN-Thomas-Rees/wiki).

## Experimental Identifiers
Experiments conducted for CNP45-ABHD14B are identified as follows:
1. First three characters are TIR (initials of the experimentalist)
2. Underscore
3. Three numbers which are the main identifier of the molecule
4. Underscore
5. Two numbers which describe the repeat/batch number

## ELN Pages
Each issue represents one experiment which are then sorted into *open* and *closed*. An *open* issue represents an experiment that is being planned or that is currently being run, while a *closed* issue represents a completed experiment. 

Issues will also be tagged with labels describing the aim and status of each experiment, descriptions of which can be found in the relevant [tab]().
